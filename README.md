# Bare Metal Driver Suite for Raspberry Pi Pico
A modular, hardware-abstraction driver suite for microcontrollers, designed around the Raspberry Pi Pico. <br/> 
Developed in C with CMake. <br/>
The pico-sdk wrapper version can be found [here](https://github.com/EYXLiu/Driver-Suite)
